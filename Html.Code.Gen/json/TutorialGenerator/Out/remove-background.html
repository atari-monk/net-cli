• Open the Nvidia broadcast app and navigate to the Camera tab.

  • Select your camera source

  • Select the resolution you want the camera to be.

  • Select effect to “Background Removal” option.

  • Add created virtual camera to Streamlabs/OBS.
    Go to sources -> Add Source -> Camera -> Select Nvidia Broadcast.
    You can resize and position this as you like.

  • Use Nvidia Broadcast Microphone Features to remove background noise.

  • Instead of removing your background you can replace it with any image you want.
